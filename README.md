The Password Generator App is a sleek, responsive web tool designed to help users create secure, customizable passwords with ease. With a modern UI and real-time strength feedback, it empowers users to generate strong passwords tailored to their preferences.

✨ Key Features
Customizable Options: Choose password length and toggle inclusion of uppercase, lowercase, numbers, and symbols.

Real-Time Strength Meter: Visual feedback on password strength (Weak, Medium, Strong) based on selected criteria.

Copy to Clipboard: Instantly copy the generated password with a single click.

Responsive Design: Optimized for both desktop and mobile devices.

Elegant UI: Clean layout with gradient backgrounds, smooth transitions, and intuitive controls.

🛠️ Technologies Used
HTML5 & CSS3: Semantic structure and stylish, responsive design using CSS variables and gradients.

JavaScript: Handles password generation logic, strength evaluation, and dynamic UI updates.

Font Awesome: Adds intuitive icons for better user interaction.

This project is perfect for learning DOM manipulation, form controls, and building interactive tools that prioritize both security and user experience
